# End-User Documentation for SmartLib

## 1. Introduction

Welcome to SmartLib, a powerful knowledge management and retrieval system. This guide is designed to help you navigate the application, understand its features, and get the most out of your knowledge base. Whether you're searching for information, uploading documents, or engaging with AI-powered chat, this guide will help you master all aspects of SmartLib.

## 2. Getting Started

### 2.1 Accessing SmartLib

SmartLib is accessible through your web browser at the URL provided by your administrator (typically https://your-organization-smartlib.azurewebsites.net). For the best experience, we recommend using the latest versions of Chrome, Firefox, Edge, or Safari.

### 2.2 User Registration and Login

#### 2.2.1 Logging In

1. Navigate to the SmartLib URL
2. On the login page, enter your username and password
3. Click "Sign In" to access the system

If your organization uses Azure Active Directory for authentication, you may see an option to "Sign in with Microsoft" instead.

#### 2.2.2 Registration (if enabled)

If self-registration is enabled:

1. Click "Register" on the login page
2. Fill out the registration form with your details
3. Submit the form to create your account
4. Wait for any required verification or approval (if configured)

#### 2.2.3 Password Recovery

If users forget the password:

1. Admin shall go to admin panel
2. click the button in list of particular users for menu reset password, and admin will see the requested to create temporary password
3. Temp password will be share to user
4. Once the user logs in with temp password and user can changes it at change password page.

### 2.3 User Interface Overview

After logging in, you'll see the SmartLib dashboard with the following main areas:

1. **Navigation Menu**: Located on the left side, providing access to all major features
2. **Quick Search Bar**: At the top of the page for easy access to search functionality
3. **Dashboard Widgets**: Showing recent activity, featured content, and quick actions
4. **User Menu**: In the top-right corner, providing access to your profile and settings

## 3. Searching for Information

### 3.1 Basic Search

The basic search functionality allows you to quickly find information across all accessible knowledge bases:

1. Enter your query in the search bar at the top of any page
2. Press Enter or click the search icon
3. Review the results sorted by relevance
4. Click on any result to view the full content

### 3.2 Advanced Search

For more precise searches, use the advanced search feature:

1. Click "Advanced" next to the search bar
2. Specify search parameters such as:
   - Knowledge base selection
   - Date ranges
   - Document types
   - Categories or tags
   - Specific fields or metadata
3. Click "Search" to execute the query
4. Use filters on the results page to further refine

### 3.3 Search Results

The search results page displays:

1. **Result Snippets**: Brief excerpts showing the context of your search terms
2. **Document Info**: Type, date, author, and other metadata
3. **Relevance Score**: Visual indicator of how closely each result matches your query
4. **Filter Panel**: Options to narrow down results further
5. **Pagination Controls**: For navigating through multiple pages of results

### 3.4 Viewing Documents

When you click on a search result:

1. The document opens in the document viewer
2. Your search terms are highlighted within the document
3. Table of contents (if available) appears on the left
4. Document actions are available in the toolbar:
   - Download
   - Share
   - Add to favorites
   - Print
   - Provide feedback

## 4. Chat with AI

### 4.1 Starting a Chat

The AI chat feature allows you to have natural language conversations about your knowledge base:

1. Click "Chat" in the main navigation menu
2. Select a knowledge base to chat about (if prompted)
3. Type your question or prompt in the chat input field
4. Press Enter or click the send icon
5. The AI will respond with information from your knowledge base

### 4.2 Chat Features

The chat interface offers several helpful features:

1. **Source Citations**: References to the specific documents used in responses
2. **Follow-up Suggestions**: Suggested questions to continue the conversation
3. **Conversation History**: Access to previous chat sessions
4. **Context Retention**: The AI remembers previous questions in the same session
5. **Document Preview**: Click on citations to preview source documents

### 4.3 Effective Prompting

To get the best results from the AI chat:

1. **Be Specific**: Clearly state what you're looking for
2. **One Topic at a Time**: Focus each question on a single topic
3. **Provide Context**: Include relevant background information
4. **Use Follow-ups**: Ask for clarification or more details as needed
5. **Try Different Approaches**: Rephrase your question if needed

### 4.4 Saving and Sharing Chats

You can save and share your chat conversations:

1. Click the "Save" icon in the chat toolbar
2. Give the conversation a name
3. Access saved chats from the "History" tab
4. Use the "Share" button to generate a link others can access (if enabled)

## 5. Uploading and Managing Content

### 5.1 Document Upload

To add new documents to the knowledge base:

1. Click "Upload" in the main navigation menu
2. Select the target knowledge base (if you have access to multiple)
3. Choose the upload method:
   - Drag and drop files onto the upload area
   - Click "Browse" to select files from your device
   - Use "URL Import" to add documents from web links
4. Add metadata (optional):
   - Categories
   - Tags
   - Description
   - Custom fields (if configured)
5. Click "Upload" to start processing

### 5.2 Supported File Types

SmartLib supports a wide range of file types, including:

- Documents: PDF, DOCX, DOC, TXT, RTF, ODT
- Spreadsheets: XLSX, XLS, CSV
- Presentations: PPTX, PPT
- Images (with text): PNG, JPG, TIFF
- Web content: HTML
- Code files: Various programming languages

### 5.3 Upload Status and History

After uploading documents:

1. A progress indicator shows the status of each upload
2. Processing status updates in real-time
3. You can continue using the system while uploads process
4. Access the "Upload History" tab to view past uploads
5. Check for any processing errors or warnings

### 5.4 Managing Your Documents

To manage documents you've uploaded:

1. Click "My Documents" in the main navigation menu
2. View a list of all documents you've added
3. Use filters to find specific documents
4. Select documents to:
   - Edit metadata
   - Move between knowledge bases (if permitted)
   - Delete documents
   - Check processing status

## 6. Working with Knowledge Bases

### 6.1 Knowledge Base Selection

If you have access to multiple knowledge bases:

1. Use the knowledge base selector in the navigation menu
2. The currently active knowledge base is highlighted
3. Your search and chat interactions will apply to the selected knowledge base
4. You can select "All Knowledge Bases" for cross-knowledge searches

### 6.2 Knowledge Base Features

Each knowledge base may offer different features:

1. **Documents**: The main collection of content
2. **Categories**: Organizational structure for documents
3. **Featured Content**: Highlighted documents or collections
4. **Analytics**: Usage statistics and insights (if enabled)
5. **Collaborative Features**: Comments, ratings, or annotations (if enabled)

### 6.3 Personal Workspace

Your personal workspace provides:

1. **Saved Searches**: Quick access to frequently used searches
2. **Favorites**: Documents you've marked as favorites
3. **Recent Activity**: Documents you've recently viewed or interacted with
4. **Notes**: Personal annotations on documents (if enabled)
5. **Notifications**: Updates about changes to content you follow

## 7. Collaboration Features

### 7.2 Document Ratings and Feedback

Provide feedback on documents to improve quality:

1. Use the rating controls (typically stars) to rate a document

## 9. Customizing Your Experience

### 9.1 User Profile Settings

To customize your profile:

1. Click your username in the top-right corner
2. Select "Profile" or "Settings"
3. Update your personal information:
   - Display name
   - Profile picture
   - Contact information
   - Password
4. Click "Save" to apply changes

### 9.2 Interface Preferences

Customize the user interface:

1. Go to "Settings > Preferences"
2. Configure display options:
   - Theme (light/dark mode)
   - Default landing page
   - Results per page
   - Default knowledge base
   - Language preference
3. Save your preferences

### 9.3 Notification Settings

Manage your notification preferences:

1. Go to "Settings > Notifications"
2. Configure which events trigger notifications:
   - Document updates
   - Comments on your content
   - Mentions
   - System announcements
3. Choose notification methods:
   - In-app
   - Email
   - Browser (if supported)
4. Set frequency preferences
5. Save your settings

## 10. Advanced Features

Create saved searches and alerts:

Use keyboard shortcuts for faster navigation:

- `Esc` - Close dialogs or panels

- `Ctrl+S` - Save current item
- `Alt+Left/Right` - Navigate history
- `Ctrl+F` - Find in document

## 11. Troubleshooting

### 11.1 Common Issues and Solutions

**Issue: Unable to log in**
- Verify your username and password
- Check Caps Lock is off
- Try the "Forgot Password" option
- Contact your administrator if issues persist

**Issue: Search returns unexpected results**
- Try more specific search terms
- Use advanced search filters
- Check knowledge base selection
- Verify document permissions

**Issue: Document upload fails**
- Check file size (typically limited to 50MB)
- Verify file type is supported
- Ensure stable internet connection
- Try a different browser

**Issue: Chat provides irrelevant answers**
- Be more specific in your question
- Provide additional context
- Try rephrasing the question
- Check that you're using the right knowledge base

### 11.2 Getting Help

If you need assistance:

1. Click the "Help" icon in the main navigation
2. Browse the help articles and FAQs
3. Use the search function to find specific topics
4. Contact support through the "Contact Support" form
5. Check the status page for any system issues
