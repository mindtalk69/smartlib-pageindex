# /frontend/ - SmartLib v2.0 React Chat

CopilotKit-powered chat interface for SmartLib.

## Development

```bash
# Install dependencies
npm install

# Start dev server (proxies to Flask on :5000)
npm run dev
```

## Build

```bash
# Production build to ./dist/
npm run build
```

## Testing

```bash
# Unit tests
npm test

# E2E tests (requires Playwright)
npm run test:e2e
```
