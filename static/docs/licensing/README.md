# SmartLib Licensing Documentation

This directory consolidates the licensing material required for distributing
SmartLib through on-premises packages, container images, or Azure Marketplace
listings.

## Directory Contents

- `LICENSE` – MIT License that governs SmartLib.
- `NOTICE.txt` – Attribution statements for key third-party components.
- `LICENSING_INFO.md` – Guidance for maintaining compliance and referencing
  licensing artefacts.
- `DEPENDENCIES_LICENSES.md` – Markdown summary of dependency licenses.
- `LICENSE_INVENTORY.csv` – Spreadsheet-friendly inventory of dependencies.
- `AZURE_MARKETPLACE_OSS_DISCLOSURE.md` – Disclosure required for Marketplace
  certification.
- `license_texts/` – Canonical license texts for referenced dependencies.

## Usage

1. Share this directory (and its static mirror) with customers because it
   contains the complete license bundle for SmartLib.
2. Update the inventory files whenever dependencies are added, removed,
   or upgraded.
3. Archive the directory for every official release to support audit requests.

For questions, contact the SmartLib compliance team at
`compliance@smartlib.ai`.
